This app allows you to interact with the present day halo catalogs for the Phat ELVIS simulation suite.

Using the text box on the top left, you can enter an SQL query to select only the subhalos of interest to you.
The full list of queryable columns are:
Column Description
host_id simulation number. Host mass decreases as this increases
id mass rank within a simulation (host halo has id of 0)
mass mass of the subhalo (Solar masses)
rs scale radius of best fit NFW profile as determined by Rockstar (kpc)
rvir virial radius (kpc)
vmax maximum circular velocity (km/s)
vx x bulk velocity (km/s)
vy y bulk velocity (km/s)
vz z bulk velocity (km/s)
x x coordinate of center (Mpc)
y y coordinate of center (Mpc)
z z coordinate of center (Mpc)
vpeak peak vmax ever obtained (km/s)
scale_vpeak scale factor of vpeak
dist distance from center (kpc)
infall scale factor of first host Rvir crossing (-1 = never crossed)
peri distance of closest approach to the host halo center (kpc)

By using the dropdown boxes, you may select different columns for the X- and Y-axes

The colored buttons allow you to download either your subquery or the entire dataset.
