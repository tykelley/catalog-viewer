This tab allows you to make some of the "standard" plots seen in many simulation papers.

Using the dropdown menu, you can select from:
Type Description
Infall CCDF of infall times
Mvir Subhalo virial mass function
Vmax Subhalo maximum circular velocity (Vmax) function
Vpeak Subhalo peak maximum circular velocity (Vpeak) function
Pericenter CDF of subhalo pericenters
